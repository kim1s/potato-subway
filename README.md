# Potato on the Subway — PRD

## 1. 서비스 개요

지하철을 타는 감자 캐릭터를 브랜드로 한, 매일 하나씩 영단어를 전달하는 학습 서비스.
날짜마다 새로운 영단어와 예문을 노출하고, 사용자들이 익명으로 해당 단어를 활용한 댓글을 남길 수 있다.

---

## 2. 핵심 기능

### 2-1. 오늘의 단어 (Daily Word)

- 오늘 날짜에 해당하는 영단어 1개를 메인 화면에 표시한다.
- 단어와 함께 한국어 뜻을 노출한다.
- 해당 날짜에 등록된 단어가 없으면 별도의 준비 중 이미지를 노출한다 (등록 예정).

### 2-2. 예문 (Examples)

- 단어당 예문은 최대 3개까지 어드민이 사전 등록한다.
- 화면에는 예문 1개만 노출한다.
- 영어 예문과 한국어 번역을 함께 보여준다.
- **Refresh** 버튼을 누르면 다음 예문으로 순환 전환한다.

### 2-3. 익명 댓글 (Potatos Talk)

- 로그인 없이 누구나 댓글을 작성할 수 있다.
- 댓글 내용과 작성 시각(시:분 AM/PM)을 노출한다.
- 입력 제한: 영어, 숫자, 특수문자만 허용. 한글 입력 불가.
- 최대 길이: 2,000자.

### 2-4. 히어로 이미지 (Hero Image)

- 메인 화면 상단에 이미지 영역을 제공한다.
- 어드민에서 미리 등록한 이미지 3개 중 1개를 랜덤으로 노출한다.
- 오늘의 단어 데이터가 없는 날은 별도의 준비 중 이미지를 노출한다 (에셋 등록 예정).

## 3. 화면 구성

```
┌─────────────────────────────┐
│      Potato on the Subway   │  ← 브랜드명
│      YYYY-MM-DD (요일)      │  ← 날짜 (클릭 시 날짜 선택)
│                             │
│       [히어로 이미지]        │  ← 랜덤 3종 중 1개
│                             │
│  TODAY'S WORD               │
│  ┌─────────────────────┐   │
│  │ Word                │   │  ← 영단어 (대형)
│  │ 한국어 뜻           │   │  ← 한국어 해석
│  └─────────────────────┘   │
│                             │
│  EXAMPLES          [Refresh]│
│  ┌─────────────────────┐   │
│  │ English example.    │   │  ← 영어 예문 (굵게)
│  │ 한국어 번역         │   │  ← 한국어 번역
│  └─────────────────────┘   │
│                             │
│  POTATOS TALK TO EACH OTHERS│
│  ┌──────────────┐ [Drop It]│  ← 댓글 입력 (영어/숫자/특수문자만)
│  └──────────────┘          │
│  ┌─────────────────────┐   │
│  │ comment text   08:00│   │  ← 댓글 목록 (내용 + 시간)
│  │ comment text   08:02│   │
│  └─────────────────────┘   │
└─────────────────────────────┘
```

## 4. 어드민 기능 (Admin)

| 기능 | 설명 |
|------|------|
| 단어 등록 | 날짜, 영단어, 한국어 뜻, 예문 최대 3개(영/한) 등록 |
| 히어로 이미지 등록 | 최대 3개 이미지 등록. 메인에서 랜덤 노출 |
| 단어 활성/비활성 | `isActive` 토글로 노출 여부 제어 |

> 어드민 API 접근은 `CONTENT_UPLOAD_SECRET` 헤더 인증으로 보호한다.

## 5. 데이터 모델

### Content (오늘의 단어)

| 필드 | 타입 | 설명 |
|------|------|------|
| `word` | String | 영단어 |
| `meaning.ko` | String | 한국어 뜻 |
| `meaning.en` | String | 영어 뜻 |
| `examples` | Array | 예문 목록 (최대 3개) |
| `examples[].en` | String | 영어 예문 |
| `examples[].ko` | String | 한국어 번역 |
| `publishDate` | String | 노출 날짜 (YYYY-MM-DD) |
| `monthKey` | String | 월 키 (YYYY-MM) |
| `order` | Number | 월 내 순서 |
| `isActive` | Boolean | 노출 여부 |

### Post (댓글)

| 필드 | 타입 | 설명 |
|------|------|------|
| `wordId` | ObjectId | 연결된 단어 ID |
| `content` | String | 댓글 내용 (최대 2,000자) |
| `ipHash` | String | 익명성 보장용 IP 해시 |
| `likes` | Number | 좋아요 수 |
| `createdAt` | Date | 작성 시각 |

## 7. API 엔드포인트

| Method | Path | 설명 |
|--------|------|------|
| GET | `/api/contents/daily?date=YYYY-MM-DD` | 오늘의 단어 조회 |
| GET | `/api/contents/month/:monthKey` | 월별 단어 목록 |
| GET | `/api/contents/:id` | 단어 단건 조회 |
| GET | `/api/posts?wordId=` | 특정 단어의 댓글 목록 |
| POST | `/api/posts` | 댓글 작성 |
